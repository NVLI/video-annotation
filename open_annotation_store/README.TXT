
-- SUMMARY --

The open_annotation_store module is used to store annotate while the video is playing as like annotation module in drupal.It displays the entire video annotations.

-- INSTALLATION --

* Install this module in sites/all/modules/custom and Enable it.

-- DEPENDANCIES --

*  Install this Plugin in sites/all/libraries.
    
    Open Video Annotation - library
    Videojs - library
    Videojs - Module

-- CONFIGURATION --

* Configure user permissions in Administration » People » Permissions:

  - Administer annotations
    Users in roles with the "Administer annotations" permission will able to see this.

* Go to any node page which contains video and annotate it. you can see the response in admin/content/video-annotation.
